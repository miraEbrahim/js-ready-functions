Arrays

Arrays in JavaScript are high-level list-like objects with a length property and integer properties as indexes.

Arrays' Common Operations

Explore how to create an array, retrieve, add, and remove items stored in an array

Arrays consist of square brackets and elements that are separated by commas.

There are multiple ways to create an array

Using Array Literal
Syntax
var arrayName = [element1, ..., elemntN];

Spaces and line breaks are not important. A declaration can span multiple lines 

var arrayName = [ 
    element1, 
    ..., 
    elementN
];

Using Array constructur

the Array constructor creates arrays differently depending on the arguments it receives

1. MORE THAN ONE ARGUMENTS:

Creates a new array with the arguments as items. The length of the array is set to the number of arguments.

Syntax
var arrayName = new Array(element1,...,elementN);
Example
var carCollection = new Array('Camaro SS','Dodge Challenger RT','Dodge Challenger SRT Demon');
2. ONLY ONE(NUMBER) ARGUMENT:

Creates an array with length set to the number. The constructor just sets the length property of the array to the given length, without setting the keys and those keys are never set and dont exist The constructor just sets the length property of the array to the given length, without setting the keys The number must be a positive integer otherwise an error will be thrown. Note that the array has no property keys apart from length..

Syntax
 var arrayName = new Array(number);
Example
 var arrayNumber = new Array(5);
 // [ <5 empty items> ] {length:5}
3. ONLY ONE(NON-NUMBER) ARGUMENT:

Creates a new array with the argument as the only item. The length of the array is set to 1.

Syntax
 var arrayName = new Array('non-number');
Example
 var arrayNumber = new Array('5');
 // ['5'] {length:1}

Using Array.of()

Array.of() will only work in browsers with ES6 support.

Array.of() behaves in a very similar fashion to Array() constructor, the only difference between Array.of() and Array is in how they handle a single integer argument passed to them

Syntax
 var arrayName = Array.of(single-element);
Example
 var arrayOf = Array.of(5); // [5]
 var arrayOfString = Array.of('Hello World');
 // [ 'Hello World' ]

Access (index into) an Array item

JavaScript arrays are zero-indexed: the first element of an array is at index 0, and the last element is at the index equal to the value of the array's length property minus 1. Using an invalid index number returns undefined.

We can access individual items in the array using bracket notation

Syntax Errors

Array elements are object properties in the same way that toString is a property, but trying to access an element of an array as "array_name.0" throws a syntax error because propertie names that begin with a digit cannot be referenced with dot notation

Example
var cars = ['Camaro SS','Challenger RT','Challenger SRT'];

//Access the first item
var firstCar  = cars[0]; //Camaro SS

//Access the last item
var lastCar = cars[cars.length - 1]; //Challenger SRT

//Access with dot notation 
var secondCar = cars.1;   // a syntax error

It's possible to quote the JavaScript array indexes as like this cars['1'] instead of cars[1]), although it's not necessary. The 1 in cars[1] is coerced into a string by the JavaScript engine through an implicit toString conversion. It is, for this reason, that '1' and '01' would refer to two different slots on the cars object and the following example could be true: 

console.log(cars['1'] != cars['01']);
// true

Loop over an Array

Loop over an array to do the same thing to every item in that array.

var cars = ['Camaro SS', 'Challenger RT', 'Challenger SRT'];
cars.forEach(function(item, index, array) {
console.log('Item Value : ', item, 'and Its index :', index);
});
// Item Value :  Camaro SS and Its index : 0
// Item Value :  Challenger RT and Its index : 1
// Item Value :  Challenger SRT and Its index : 2

We have two methods that add items to an array, "push()" to the end of an array and "unshift()" to the front of an array

push(): Add to the end of an Array

The push() method adds one or more elements to the end of an array and returns the new length of the array

The push method relies on a length property to determine where to start inserting the given values. If the length property cannot be converted into a number, the index used is 0. This includes the possibility of length being nonexistent, in which case length will also be created

push()

Syntax
 arrayName.push(element1[, ...[, elementN]]);
Parameters

elementN

The elements to add to the end of the array.

Return value

The new length property of the object upon which the method was called

push() doesnt return a new array 

  var sports = ['soccer', 'baseball']; 
  var total = sports.push('football', 'swimming');
  console.log(sports); // ['soccer', 'baseball', 'football', 'swimming']
  console.log(total);  // 4
